# DXF to Lua Converter

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Platform](https://img.shields.io/badge/Platform-Windows-blue)](https://github.com)

Программа для извлечения замкнутых контуров из DXF файлов и преобразования их в Lua код. Поддерживает линии и арки, автоматически определяет замкнутые контуры и генерирует код в требуемом формате с визуализацией результатов.


## Как это работатет

Начертите в CAD контур(ы), и сохраните в формате DXF.
Требования к контурам:
- Контуры должны состоять только из линий и дуг
- Контуры должны быть замкнутые
- Полилинии надо разбить на примитивы (линии и дуги)
- Прочие типы примитивов не обрабатываются

После загрузки полученного DXF в dxf2lua, получите:
- Отчет по обработанному файлу
- Изображение найденных контуров
- Lua код для построения найденных контуров

Полученные описания контуров можно копировать и использолвать в своих lua скриптах
